🎉 Explorando addEventListener

Em JavaScript, o método addEventListener permite escutar eventos, como cliques, teclas e movimentos. Ele nos dá controle total para tornar páginas mais interativas e dinâmicas.

🧪 Exemplo básico:


const btn = document.getElementById("btn");
btn.addEventListener("click", () => {
  alert("Clique detectado!");
});
😄 Você descobriu uma mensagem secreta!
